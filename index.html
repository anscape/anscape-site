ダイヤの乱れで生まれる“詰まり（STUCK）”を、次の行動（ACTION）に変える。

Anscapeは、乱れた状況で「いま何が起きていて、どう動くべきか」を最短で理解し、 左右の画面で 移動／滞在 を比較して、迷わず決めるためのアプリです。

App Store（サンプル） → 使い方を見る →

解決する課題

  • 予期しない遅延・運休で、次の行動が取れず立ち止まってしまう
  • 係員・掲示板・複数アプリを往復し、情報探索に時間を溶かす
  • 結果として、焦る／無駄に移動する／待つだけになる

Anscapeの価値

  • 1タップで“いま起きていること”を把握（周辺路線・列車）
  • 判断を2択に圧縮：A 振替 / B 滞在・移動
  • 左右比較で“決める”を支援し、そのまま実行へつなぐ

主な機能

Fact

周辺路線の運行状況

現在地付近の駅を起点に、複数路線の状況を一望して把握します。

Trains

近接列車の状況

選んだ路線について、近くを走る列車の状況を確認できます。

Action A

振替する（即効性）

  • シェアサイクルマップ（例）
  • 乗換案内サービスへ遷移（例：Yahoo乗換案内）
  • 地図サービスへ遷移（例：Googleマップ）
Action B

滞在・移動（比較して決める）

  • カフェ / コワーキング / シェアサイクル / タクシー / ご飯 など
  • 候補の詳細確認 → 各サービスへ遷移

想定ユーザー

  • 都市部（特に東京圏）の通勤・通学/移動中ユーザー
  • 駅・路線の近くで“次の行動”を素早く決めたい方
  • 普段よく使う駅の情報を集めたい老若男女の乗客
